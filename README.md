# 모비노기 모바일 매크로

마비노기 모바일 Windows 전용 자동화 도구 모음입니다.

현재 이 저장소에는 세 가지 프로그램이 들어 있습니다.

| 프로그램 | 설명 | 폴더 | 릴리스 파일 |
| --- | --- | --- | --- |
| `Autosell` | 인벤토리에서 미리 등록한 재료 아이템을 자동으로 찾아 판매 | `Autosell/` | `Autosell.exe` |
| `Processing` | 가공 재료를 자동으로 제작하고, 여러 재료를 순환 처리 | `Processing/` | `Processing.exe` |
| `Fishing` | 집중해서 낚시하기를 자동화 | `Fishing/` | `Fishing.exe` |

---

## 개요

### Autosell

- 판매할 아이템을 직접 설정 한 뒤 자동 판매를 반복합니다.
- 여러 품목을 설정 할 수 있습니다.
- 시작 시 게임 창 크기를 한 번 자동으로 맞춥니다.
- 오류 상황에서 빠져나오기 위한 복구 로직이 포함되어 있습니다.

### Processing

- 가공 재료를 자동 인식해 제작을 반복합니다.
- 여러 재료를 동시에 선택하면 제작대를 전환하며 로테이션합니다.
- 재료 부족이나 주문 가득 참 같은 상황을 감지해 다음 작업으로 넘어갑니다.

### Fishing

- 집중해서 낚시하기를 자동으로 합니다
- 미끼를 이용하시면 효율이 극대화 됩니다.

---

## 사용 방법

일반 사용자는 GitHub 소스코드가 아니라 Releases에서 실행 파일을 받으면 됩니다.

1. [Releases](../../releases) 페이지로 이동합니다.
2. 최신 버전에서 필요한 파일을 다운로드합니다.
> 💡 보안 경고가 뜨면 **"추가 정보" → "실행"** 을 눌러주세요. (바이러스 아니에요!)

---

## 공통 주의사항

- Windows 10 / 11 환경만 지원합니다.
- 이미지 스캔 방식으로 동작하므로 게임 창이 다른 창에 가려지면 안 됩니다.
- 매크로 사용 시 <b>창 크기 맞추기 버튼</b>을 클릭하여 사용하시고, 매크로가 실행되는 동안에는 게임 창 크기를 변경하지 마세요.
- 매크로 실행 시 표시되는 가이드를 먼저 읽고 시작하세요.

## 프로그램별 실행 전 준비

### Autosell

- 딱히 없음!

### Processing

- 게임 창은 창 크기 맞추기 버튼을 눌러 꼭 맞춰주세요.
- <b>중요!! 제작대에 가서 가공 창을 열어둔 상태에서 시작해야 합니다.</b>
- 여러 재료를 선택한 경우에도 선택한 재료 중 하나의 제작대를 열어둔 채 시작하면 됩니다.
- 각 분야에서 하나씩만 선택할 수 있으며 최대 4개까지 동시에 운용합니다.

### Fishing

- 원하는 낚시터에 간 뒤, 앞으로 가도 (w를 눌러도) 캐릭터가 움직이지 않는 곳에서 시작해야 합니다.
- 매번 낚시 할 때마다 아주 조금씩 움직여서 시간이 지나면 낚시터를 벗어날 가능성이 있습니다.
- 집중해서 낚시하기를 선택해야 하고, 보유중인 펫은 꼭 넣어주세요 (펫 상호작용 때문에..)
