# 모비노기 통합 매크로

마비노기 모바일의 **채집/낚시** 및 **가공** 을 자동화하는 매크로입니다.

> ⚠️ **Windows 전용**입니다. (이미지 스캔 방식으로 동작)

---

## 📦 프로그램 구성

| 프로그램 | 설명 | 폴더 |
|---------|------|------|
| **Gathering** | 마력 깃든 돌 / 반짝이는 이끼 채집 + 자동판매, 낚시 | `Gathering/` |
| **Processing** | 가공 자동화 (철괴, 목재, 가죽, 옷감 등) | `Processing/` |

---

## 🚀 사용 방법

[Releases](../../releases) 페이지에서 **최신 버전을 다운로드**합니다.
> 💡 보안 경고가 뜨면 **"추가 정보" → "실행"** 을 눌러주세요. (바이러스 아니에요!)

## 🎯 매크로별 사용법

### Gathering (채집/낚시)

| 모드 | 설명 | 위치 |
|------|------|------|
| 마력깃든돌 | 광맥 자동 채집 + 백동광석/돌멩이 자동 판매 | 얼음협곡 광맥 |
| 반짝이는이끼 | 버섯 자동 채집 + 튼튼버섯/포자 자동 판매 | 시드스넷타 버섯 |
| 낚시 | 집중해서 낚시하기 자동화 | 원하는 낚시터 / 앞으로 가도 안 움직이는 자리 |

**실행 전 세팅:**
1. 게임 창 크기를 **1280×960**으로 설정 후 **세로만 최소**로 줄이기
2. 인벤토리 열기 → **아이템 탭** 클릭 → 닫기
3. 해당 위치로 이동 후 매크로 시작

---

### Processing (가공)

4종류의 작업대에서 총 **12가지 재료**를 자동 가공합니다:

| 작업대 | 재료 |
|--------|------|
| 금속 | 철괴, 강철괴, 합금강괴 |
| 목재 | 목재, 목재+, 상급목재 |
| 가죽 | 가죽, 가죽+, 상급가죽 |
| 옷감 | 옷감, 옷감+, 상급옷감 |

**참고 사항:**
- ✅ **복수 선택** — 여러 재료를 동시에 체크하면 자동으로 작업대를 전환하며 로테이션
- ✅ **반드시 체크한 작업대에 가서 작업대를 연 상태로 실행**

**실행 전 세팅:**
1. 게임 창 크기를 **1280×960**으로 설정 후 **세로만 최소**로 줄이기
2. 선택할 재료의 **아무 작업대에 가서 가공 창을 열어놓기**
   <br>ex) 철괴, 가죽을 체크했을 경우 금속 제작대 or 가죽 제작대에 가서 가공창 열기
4. 매크로에서 재료 체크 후 시작
5. 제작 재료 채집은 자동화되어 있지 않으니, **재료를 미리 구비해주세요**
6. 철괴의 경우 광석이 아닌 **철광석으로만** 가공 됩니다.

---

## ⚠️ 공통 주의사항

- **이미지 스캔 방식**으로 동작하기 때문에 게임 창이 다른 창에 **가려지면 안 됩니다**
- 게임 창 크기는 반드시 **1280×960** → **세로만 최소**로 설정해주세요
